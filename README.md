    <h2>📌 Project Overview</h2>
    <p>
        This Power BI dashboard analyzes inventory performance and stock health.
        It provides key business metrics to monitor sales performance, stock exposure,
        and inventory efficiency for better decision-making.
    </p>

    <h2>🛠 Tools Used</h2>
    <ul>
        <li>Power BI</li>
        <li>DAX (Data Analysis Expressions)</li>
        <li>Data Modeling</li>
    </ul>

    <h2>📊 Key Performance Indicators</h2>
    <ul>
        <li>Total Sales</li>
        <li>Total Profit</li>
        <li>Total Quantity</li>
        <li>Inventory Turnover</li>
        <li>Growth Percentage</li>
        <li>Low Stock Percentage</li>
    </ul>

    <h2>📈 Dashboard Features</h2>
    <ul>
        <li>Top 5 Products by Total Sales</li>
        <li>Low Stock Exposure by Category</li>
        <li>Inventory Health Overview (Donut Chart)</li>
        <li>Monthly Revenue Performance Trend</li>
        <li>Business Insight Summary Section</li>
    </ul>

    <h2>🔎 Business Insights</h2>
    <p>
        The dashboard highlights inventory turnover to measure stock movement efficiency.
        It identifies low stock exposure risk areas and supports strategic replenishment decisions.
    </p>
